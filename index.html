Welcome to Statlas!

Statlas is a new and fun way to learn about countries from around the world, simply input the country you want to learn about and we will provide you with the country flag and some stats.

When you're ready to search, use the below search bar, type in the country and hit submit. This will then either return the country information or return you a "not found" message. Countries must be spelt correctly.
